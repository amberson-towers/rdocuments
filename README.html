0.1 rdocuments

This repo contains an .RProj directory created and pushed from a local RStudio .RProj directory on Passmore’s Mac Book Pro. The .Rmd files on the repo are RMarkdown files that simply are Markdown files (familiar as .md) with some YAML code inserted in headers for document control. The .Rmd files are text files that you may edit in any text editor, if you are not an R and RStudio user.

bookdown, an R package generated the layout and other document control for this document, which displays three individual documents that specify the governance of Amberson Towers Condominium on 5 Bayard Road in Pittsburgh, Pennsylvania USA 15213.

GitHub pages are served as a web site from the ‘docs’ directory in the repo. The public-facing presence for the Amberson documents is https://amberson-towers.github.io/rdocuments/.
